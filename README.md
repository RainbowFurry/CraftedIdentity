# 🪪 CraftedIdentity

Ein modernes, anpassbares Minecraft-Profil-Plugin für Roleplay-Server und Community-SMPs – erstelle dein eigenes Profil mit Name, Alter, Geschlecht, Sexualität, Beschreibung, Socials, Land und Sprachen!

## ✨ Features

- 🪪 Personalisiertes Spielerprofil
- 🏷️ Echter Name, Alter, Geschlecht, Sexualität, Beschreibung
- 💬 Social-Media-Links (Discord, Twitter/X, Instagram, Twitch, YouTube – erweiterbar!)
- 🌍 Heimatland angeben
- 🗣️ Beliebig viele Sprachen hinzufügen
- 📊 Andere Spielerprofile anschauen
- 🎨 Komplett konfigurierbar (Menüs, Items, Farben!)
- ⚡ Schnell und leicht
- 💾 Speicherung als JSON
- 🔮 Später: Datenbank-Unterstützung (MySQL/MongoDB)

# 📖 Commands

| Command | Description |
|---|---|
| `/profile` | Öffnet dein persönliches Profil-Menü. |
| `/profile <player>` | Zeigt das Profil eines anderen Spielers an. |
| `/profil` | Alias für `/profile`. |

# 🔑 Permissions

| Permission | Description | Default |
|---|---|---|
| `craftedidentity.command.profile` | Erlaubt die Nutzung von `/profile`. | `true` |

# 🚀 Installation

1. Lade die neueste Version herunter.
2. Platziere die `.jar`-Datei in den `plugins/`-Ordner deines Servers.
3. Starte den Server neu.
4. Konfiguriere das Plugin nach deinen Wünschen in `plugins/CraftedIdentity/config.yml`.
5. Viel Spaß!
